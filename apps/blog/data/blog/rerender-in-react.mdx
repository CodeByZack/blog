React 组件的生命周期大概是这样：

`mount -> init-render -> re-render -> unmount`

1. props 变动，引起的渲染

2. context 变动，引起的渲染

3. 父组件渲染引起的子组件重新渲染

4. 组件 unmount -> mount 引起的渲染

5. state 变动，引起的渲染

   自定义hook，三方状态管理
